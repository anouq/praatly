Welcome!

What is this?

Praatly is an app that reads Praat sound objects and visualizes the results using Plotly.

How does it work?

Record Praat sounds and save them to the object list. Download the Praat script and run it in Praat via Praat > Open Praat script > Run > Run. The result will be in the folder where the script is. Finally, upload the generated files to Praatly.

The results are displayed by type, corresponding to the sound object name.

This app requires at least Praat 6.0.40.
